Relational Database

a Relational Database is a database that organizes information into one or more tables.
a Table is a collection of data organized into rows and columns.
One can think on a table as relations
. a Column is a set of data values of a particular type
. Considering the above example on employee table , the colums will be Name, ID and Salary.
a row is a sigle record in a table that contain all the column values such as Name Marco, ID 0493876 and the Salary is 18.000.
. All the data stored in a relational database is of a certain data type.
. Some of the data types are:
Integer , a positive or negative whole number.
Text , text string.
Date , date formateed as YYYY-MM-DD for the year , month and day.
